# KOSPI_Prediction


## PROJECT

📈 KOSPI의 데이터셋을 이용해 Open(시가), Close(종가), High(고가), Low(저가), Volume(거래량) 열을 크롤링하여 구축하였다. 
그 중 분석에 사용한 열은 Close(종가) 데이터이며 이를 기반으로 시계열 분석을 진행한 주가 등락 예측 프로젝트이다.
